== Description ==

CodeSpreader allows you to inject custom CSS and JavaScript globally across your site or individually on specific posts and pages. Use the settings panel for global code, or add per-page scripts directly from the editor.

**Features:**
* Add global CSS across all pages
* Add global JavaScript across all pages
* Add custom CSS/JS per page or post from the editor
* Simple and clean settings page
* Lightweight and optimized
* WordPress coding standards compliant

== Installation ==

1. Upload the `codespreader` plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to `Settings > CodeSpreader` to start adding global CSS and JavaScript.
4. For per-page scripts, open any post or page and scroll to the CodeSpreader editor box.

== Frequently Asked Questions ==

= Can I add both CSS and JavaScript? =
Yes, both global and per-page scripts are supported.

= Will my changes persist through theme updates? =
Yes. CodeSpreader is independent of theme files.

= Is the per-page injection available for custom post types? =
Yes, it works for posts, pages, and

 registered custom post types.
